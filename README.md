# d3-chord2

d3 plugin to provide minor extensions to d3-chord such as grouped arcs

## Installing
...
```html
<script src="https://d3js.org/d3.v4.min.js"></script>
<script src="d3-chord2.js"></script>
<script>

var chord = d3.chord2();

</script>
```
